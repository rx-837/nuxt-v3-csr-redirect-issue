# nuxt-v3-csr-redirect-issue

## Setup

Make sure to install dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Start

Build the application:

```bash
# npm
npm run build

# pnpm
pnpm build

# yarn
yarn build

# bun
bun run build
```

Run the application:

```bash
# npm
npm run preview

# pnpm
pnpm preview

# yarn
yarn preview

# bun
bun run preview
```

## Issue reproduction steps

1. Build the application

1. Run the application

1. Open the application in a browser, for example, at http://localhost:3000

1. Click the link named "test"


The page url will change, but page does not render. If reload page in browser it will render.
